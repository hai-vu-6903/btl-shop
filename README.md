**A. Cài đặt Database (XAMPP)**
- Mở phpMyAdmin
- Tạo database tên: btl_webbanhang
- Import file: btl_webbanhang_full.sql

**B. Cài đặt Backend**
- cd backend
- npm install
- node server.js
=> Truy cập: http://localhost:5000

**C. Cài đặt Frontend**
- cd frontend
- npm install
- npm run dev
- => Truy cập: http://localhost:5173
